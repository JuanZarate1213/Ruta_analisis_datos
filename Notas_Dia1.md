cat: Escupe todo el contenido del archivo en la pantalla. Úsalo solo para archivos 
pequeños.

head -n 20: Te muestra solo las primeras 20 líneas de un archivo de datos. Vital para 
entender la estructura de un dataset sin abrirlo todo.

tail -n 20: Te muestra las últimas 20 líneas. Ideal para ver los registros más recientes 
en un log de servidor.

grep: El buscador maestro. Si tienes un archivo con 1 millón de líneas y buscas la
 palabra "Error" o "Laptop", usas: grep "Laptop" ventas.csv.

top o htop: Te muestra qué programas están consumiendo la memoria y el procesador de tu
 computadora/servidor.

history: Te muestra todos los comandos que has escrito antes. Si ayer hiciste algo que
 funcionó y no te acuerdas qué fue, búscalo aquí.

git add: para poner el archivo en el área de preparación, si se quiere subir todo lo 
que se ha hecho de un solo golpe se hace con un "git add .", el punto significa
"incluye todo lo que se ha hecho"
