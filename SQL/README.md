# 📊 Ruta de Análisis de Datos - SQL & Python

Este repositorio documenta mi proceso de formación técnica en análisis de datos, enfocado en el dominio de SQL (SQLite) y la gestión de entornos de desarrollo profesionales.

## 🚀 Estado del Proyecto
Actualmente en fase de **Fundamentos de SQL**, consolidando conceptos de filtrado, agregación y relaciones entre tablas (Joins).

## 🛠️ Tecnologías y Herramientas
- **Motor de Base de Datos:** SQLite 3
- **Editor:** VS Code + Extensiones SQLTools
- **Terminal:** Warp
- **Control de Versiones:** Git & GitHub

## 📂 Estructura del Proyecto
- `/SQL`: Scripts de creación, carga y consultas de práctica.
    - `01_crear_y_cargar.sql`: Estructura inicial y manejo de tipos de datos.
    - `02_relaciones.sql`: Implementación de llaves foráneas y Joins académicos.
- `/Data`: Archivos `.db` locales (ignorados en el historial mediante .gitignore).

## 🧠 Conceptos Dominados
- [x] Tipado de datos en SQLite (TEXT, INTEGER, REAL).
- [x] Orden de ejecución de consultas (FROM -> WHERE -> GROUP BY -> HAVING -> SELECT).
- [x] Diferencia técnica entre Motores Cliente-Servidor vs. Embebidos.
- [x] Normalización básica y llaves foráneas.
